EUR PREDICTOR
An AI-powered web application for predicting Estimated Ultimate Recovery (EUR) of hydraulically fractured wells using an Artificial Neural Network (ANN) model. The system supports single-well prediction, batch upload, sensitivity analysis, and completion parameter optimization — all grounded in real field data from unconventional reservoirs.
Model Overview
The ANN model takes 11 geological and completion engineering features as input and predicts EUR in bcf. Features are preprocessed using log transforms and quantile normalization before inference. The optimizer uses SLSQP / Differential Evolution (scipy) to find completion parameters that maximize predicted EUR given fixed geological conditions.
